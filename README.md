# Jogo de Adivinhação em Java

Um jogo simples de console onde o jogador deve adivinhar um número aleatório entre 0 e 100 em até 5 tentativas.

---

## Como funciona

- O programa gera um número aleatório.
- O jogador tem **5 tentativas** para acertar.
- A cada erro, o sistema informa se o número correto é **maior ou menor**.
- Se acertar, vence.
- Se acabar as tentativas, recebe **Game Over** com o número correto.

---

## Conceitos aplicados

- Estrutura de repetição (`while`)
- Estruturas condicionais (`if/else`)
- Classe `Random`
- Classe `Scanner`
- Controle de tentativas
- Validação de entrada

---

## Tecnologias

- Java
